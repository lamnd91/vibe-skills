Simple Commands

Intuitive CLI that gets out of your way

Initialize Project

vibe-skills init

List Available Skills

vibe-skills list

Install a Skill

vibe-skills install code-reviewer

Install from Config

vibe-skills install

Install by Stack

vibe-skills install --stack dotnet

Search Skills

vibe-skills search "database"

Update Skills

vibe-skills update

Remove a Skill

vibe-skills remove code-reviewer

Self Update CLI

vibe-skills self-update
